# Quick Start (3 Steps)

## Step 1: Install
```bash
npm install
```

## Step 2: Run
```bash
npm run dev
```

## Step 3: Open Browser
Go to **http://localhost:5173**

---

## What You'll See

### Monaco Editor Playground
- **6 Configuration Tabs** - Basic, Display, Editing, Suggestions, Advanced, Layout
- **55 Controls** - Sliders, dropdowns, checkboxes for every option
- **9 Languages** - JavaScript, TypeScript, Python, Java, JSON, N1QL, SQL++, Shell, Plain Text
- **Live Preview** - Changes apply instantly

### Search Workbench
- **JSON Editor** for Couchbase FTS queries
- **Hover Documentation** - Hover over any keyword for help
- **Autocomplete** - Type `"` for suggestions
- **Validation** - Real-time error checking
- **Query Execution** - Run and see results

---

## Try This First (30 seconds)

1. Click **Display** tab
2. Set **Cursor Style** = "Block"
3. Set **Line Numbers** = "Relative"
4. Watch the editor transform!
